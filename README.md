# 智词引擎 · 同程旅行关键词智能分类平台

> 基于规则的关键词分类引擎，支持旅行行业 6 大业务线 + 30+ 意图词包智能分类

## 功能特性

### 核心能力
- **一级分类**：国内酒店、国际酒店、国内机票、国际机票、景区、火车票、待确认
- **二级分类**：30+ 意图词包（城市酒店词、品牌词、目的地词、价格敏感词等）
- **实体识别**：支持城市/国家/POI 缩写的自动识别

### 数据分析
- 📊 分类分布可视化（饼图/柱状图）
- 💰 多维数据分析（花费/曝光/点击/效率）
- 🔀 交叉分析（业务线 × 词包热力图）
- ☁️ 关键词词云

### 词包管理
- 🎯 智能筛选（按业务线/词包/城市）
- 📦 Jenks 自然断点法自动分档
- 📥 一键导出 Excel

## 快速开始

### 方式一：直接使用

```bash
# 使用任意静态服务器
npx serve .
# 或
python -m http.server 8080
```

然后浏览器访问 `http://localhost:8080`

### 方式二：直接打开（部分功能受限）

```bash
# Windows
start index.html

# macOS
open index.html

# Linux
xdg-open index.html
```

> ⚠️ 直接打开时，图表和词云需要网络加载 CDN 依赖；建议使用方式一

## 使用指南

### 输入关键词

**粘贴模式**：每行一个关键词，点击"开始分类"

```
上海酒店推荐
北京故宫门票
五一机票特价
杭州旅游攻略
```

**上传模式**：上传 Excel 文件，支持多列配置

| 列 | 说明 |
|---|---|
| 第1列 | 关键词（必填） |
| 第2列 | 花费（可选） |
| 第3列 | 曝光（可选） |
| 第4列 | 点击（可选） |

### 筛选与导出

- 使用筛选器精确定位关键词
- 支持按国家/城市二次筛选
- 点击"导出 CSV"下载分类结果

### 词包生成

1. 选择业务线和意图词包范围
2. 选择分档维度（花费/曝光/点击）
3. 点击"智能生成词包"自动分档

## 技术架构

```
keyword-classifier/
├── index.html          # 主页面
├── css/
│   └── style.css       # 样式表
└── js/
    ├── dicts.js        # 词库定义（城市/品牌/意图词包）
    ├── classify.js     # 分类引擎 + 正则预编译
    └── ui.js           # UI 渲染 + 交互逻辑
```

### 分类流程

```
输入关键词
    ↓
预编译正则匹配 → L1 一级分类（酒店/机票/景区/火车票）
    ↓
L2 二级分类（品牌词/目的地词/价格敏感词...）
    ↓
实体识别（城市/国家）
    ↓
结果输出
```

## 词库说明

### L1 一级分类

| 分类 | 关键词特征 |
|------|-----------|
| 国内酒店 | 城市+酒店、民宿、客栈、公寓等 |
| 国际酒店 | 含国际城市名、免签、境外等 |
| 国内机票 | 城市+机票、航班、高铁等 |
| 国际机票 | 出国机票、国际航班等 |
| 景区 | 门票、景点、旅游攻略等 |
| 火车票 | 高铁、火车、抢票等 |

### L2 二级词包

**酒店类**：品牌词、竞品词、连锁酒店词、城市酒店词、目的地词、场景词、即时需求词、价格敏感词、品质高端词、地理位置词、节点词、长尾攻略词、通用词

**机票类**：品牌词、竞品词、航司词、目的地词、航段词、产品词、比价决策词、紧急出行词、人群词、节点词、业务场景词、查询导航词、业务费用词

**景区类**：品牌词、竞品词、票务意图词、评价攻略词、主题乐园、自然风光、人文历史

**火车票类**：品牌词、竞品词、抢票候补词、通勤词、旅游出行词、场站词
